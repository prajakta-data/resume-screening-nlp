# Resume Screening using NLP

This project implements an NLP-based resume screening system that classifies resumes into job categories and ranks candidates using skill-based scoring.

## Features
- Resume text cleaning and preprocessing
- TF-IDF vectorization
- Multi-class classification with Logistic Regression
- Hyperparameter tuning with GridSearchCV
- Skill-based ranking system
- Confusion matrix visualization

## Dataset
Kaggle Resume Dataset:
https://www.kaggle.com/datasets/gauravduttakiit/resume-dataset

Dataset is not included due to size and licensing.

## Tech Stack
- Python
- Pandas & NumPy
- Scikit-learn
- Matplotlib & Seaborn
- NLP (TF-IDF)

## How to Run
1. Clone repository
2. Place dataset in `/data` folder
3. Run notebook

## Results
- Macro F1-score: 99.45%
- Accuracy: 99.48%
